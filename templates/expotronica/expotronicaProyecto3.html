Bot De Whatsapp

Es un proyecto en el que se genera un bot dentro de whatshap web el bot responderá automáticamente a cada mensaje que te llegue a tu numero de teléfono instantáneamente, cada mensaje conlleva una respuesta, ademas tiene varias funciones como poder jugar al preguntados pedirle que ponga música simplemente con un comando.

¿Como Funciona?

El bot funciona usando Python y una libreria llamada pyautogui, con esta libreria dependiendo el color de una cierta posicion detecta si llego un mensaje, ahi copia el mensaje y lo guarda para poder buscar la mejor respuesta y poder enviarsela al usuario.
